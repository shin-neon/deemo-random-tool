# DEEMO -Reborn- 非公式補助ツール v0.99

スマホ操作前提の DEEMO -Reborn- 抽選用静的Webアプリです。

## 使用方法

### 抽選

1. 下部メニューで「抽選」を開きます。
2. パック条件を選びます。
   - 「全パック」: 対象パック全体から抽選します。
   - 「パックを選択」: 表示されたパック一覧から複数選択して抽選します。
3. 難易度条件を選びます。
   - 「全て」: Easy / Normal / Hard 全てから抽選します。
   - 「難易度を選択」: Easy / Normal / Hard から複数選択して抽選します。
4. 最小Lv、最大Lvを 1〜11 の範囲で指定します。
5. 「抽選」ボタンを押すと、条件に合う曲から1曲表示されます。
6. 「twitchチャット用に結果コピー」を押すと、Twitchチャットに貼りやすい形式で結果をコピーできます。

コピーされる形式:

```text
[抽選結果] 曲名 / アーティスト名 / 難易度 Lv
```

### 楽曲検索

1. 下部メニューで「楽曲検索」を開きます。
2. 検索欄で、曲名・アーティスト名・Lvを検索できます。
3. 「パック表示」から、全パック表示またはパック別表示を選べます。
4. 各曲カードには、サムネイル、曲名、アーティスト名、パック名、Easy / Normal / Hard のLvが表示されます。
5. スマホで曲カードをタップすると、以下形式で曲情報をコピーできます。

```text
曲名 / アーティスト名 / 難易度 Lv
```

### 設定保存

抽選条件と曲一覧のパック表示は、ブラウザの localStorage に保存されます。次回同じ端末・同じブラウザで開いたときに、前回の設定が復元されます。

## ファイル構成

```text
index.html
style.css
script.js
songs.js
pack_image/
```
## 注意

- `pack_image` の画像ファイル名は `songs.js` の `packs` と一致させています。
- 今回はTVモード難易度のみ実装しています。
- VRモード難易度は未使用です。
